# Security

## Supported Content

This repository contains public showcase documentation and static HTML only. It is not the operational cut-on-site wall and conventional pitched roof framing material build-list workspace.

## Sensitive Material Policy

Do not commit:

- real site addresses, client details, drawing numbers tied to real projects, plans, PDFs, DWG/DXF files, ZIP files, workbooks, CSV exports, photos, logs, or generated outputs;
- production source code, private templates, private fixtures, or internal scripts unless separately approved for public release;
- `.env` files, API keys, tokens, service credentials, database details, or deployment configuration;
- private git history from internal workspaces or private monorepos.

## Reporting A Concern

If you find sensitive material in this repo, open a GitHub security advisory if available. If not available, open an issue with a minimal description and do not paste secrets or personal information into the issue body.
