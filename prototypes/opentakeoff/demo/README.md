# Demo plans

Two sample plans ship with OpenTakeoff so you can try a takeoff in seconds —
no account, no upload. In the app, click **Load sample plan** on the empty
canvas (it opens `sample-finish-plan.pdf`), or drag either file in yourself.

## `sample-finish-plan.pdf` — what the "Load sample plan" button opens

A real **floor finish plan + room finish schedule** (2 sheets) from a U.S.
Department of Veterans Affairs medical-center renovation — a publicly bid
federal construction set. It shows flooring the way a real takeoff does:
carpet tile (CPT-1/CPT-2), VCT, luxury vinyl, porcelain tile, resinous epoxy,
rubber, and welded-seam sheet, called out room-by-room with a matching finish
schedule. The drawn scale (`1/8" = 1'-0"`) auto-detects.

1. `npm run dev` in `../web`, open the app, click **Load sample plan**
2. The scale auto-detects; pick a condition (e.g. `CPT-1`) and click **One-Click
   Area** inside a room, or trace one with **Area**
3. Add a waste %, open **Report**, and export your quantities + buy list

> Source: a federal VA solicitation drawing set (publicly released for bidding).
> Document metadata stripped; included only to demonstrate the tool.

## `sample-plan.pdf` — a tiny synthetic sheet

A **fully original, synthetic** floor plan — a tabloid sheet with four enclosed
rooms, room labels, a sheet number (`A-101`), and a drawn scale note
(`1/4" = 1'-0"`). No real project, no copyright. Generated by
[`make_sample_plan.py`](make_sample_plan.py) (pure Python, no dependencies) so
it's reproducible and auditable:

```bash
python3 make_sample_plan.py   # rewrites sample-plan.pdf
```
